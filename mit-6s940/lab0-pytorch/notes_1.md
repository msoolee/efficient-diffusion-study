# Lab 0 - PyTorch 실습 정리

이 실습은 MIT 6.S940 Efficient AI 수업의 Lab 0 과제로,
PyTorch의 기본적인 사용법(Tensor, DataLoader, Module 등)과  
CIFAR10 데이터셋을 활용한 모델 구현 및 시각화를 다룹니다.

## 🧠 주요 내용

- CIFAR10 데이터셋 불러오기 및 전처리
- 시각화를 통한 클래스별 샘플 확인
- VGG 아키텍처 구현 및 세부 주석
- BatchNorm, MaxPool, AvgPool의 실제 동작 이해
